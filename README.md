# Sidebar Menu

Menu lateral responsivo com submenus expansíveis, feito com HTML, CSS e JavaScript puro.

## Funcionalidades

- Sidebar com botão para abrir e fechar.
- Submenus em formato accordion para `Create` e `Todo-List`.
- Ícones e texto alinhados corretamente no mesmo link.

## Estrutura

- `index.html`: marcação da sidebar e conteúdo principal.
- `style.css`: estilos do layout, do menu e das animações.
- `script.js`: lógica de abrir/fechar sidebar e submenus.

## Como usar

1. Abra `index.html` no navegador.
2. Clique no botão do topo para recolher ou expandir a sidebar.
3. Clique em `Create` ou `Todo-List` para abrir e fechar os submenus.

## Observação

O projeto usa JavaScript simples, sem dependências externas.
